# AI4Alzheimers

Modularized deep learning project for Alzheimer's severity classification.

## Structure
- `src/`: Core logic (model, dataset, utils).
- `scripts/`: Training and inference scripts.
- `models/`: Saved model checkpoints.

## Usage
1. Train:
   ```bash
   python scripts/train_kaggle.py
   ```
2. Inference:
   ```bash
   python scripts/inference.py
   ```
